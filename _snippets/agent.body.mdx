<ParamField body="getContextFromCallback" type="boolean" default="false">
  Whether to get the retrieve the agent's context from the callback URL prior to
  starting the call. If true, the context parameter will be ignored. You can use
  this to customize the agent's context based on the caller's phone number, or
  to retrieve the context from a database.
</ParamField>

      This is the URL for the call recording.

  </Expandable>
